
## –ß—Ç–æ —Ç–∞–∫–æ–µ if-elif-else

–£—Å–ª–æ–≤–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä **if-elif-else** –≤ Python ‚Äî —ç—Ç–æ —Å–ø–æ—Å–æ–± –Ω–∞–ø–∏—Å–∞—Ç—å –ø—Ä–æ–≥—Ä–∞–º–º–Ω—ã–π –∫–æ–¥ —Ç–∞–∫, —á—Ç–æ–±—ã –æ–Ω –≤—ã–¥–∞–≤–∞–ª —Ä–µ–∑—É–ª—å—Ç–∞—Ç –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ç–æ–≥–æ, –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–µ —É—Å–ª–æ–≤–∏–µ –∏–ª–∏ –Ω–µ—Ç.

```python
    # –±–ª–æ–∫ –∫–æ–¥–∞, –∫–æ—Ç–æ—Ä—ã–π –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –µ—Å–ª–∏ –Ω–∏ –æ–¥–Ω–æ –∏–∑ —É—Å–ª–æ–≤–∏–π –Ω–µ –∏—Å—Ç–∏–Ω–Ω–æ

```

## –£—Å–ª–æ–≤–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä if

–í —è–∑—ã–∫–µ Python –≤—ã—Ä–∞–∂–µ–Ω–∏–µ —Å—á–∏—Ç–∞–µ—Ç—Å—è –∏—Å—Ç–∏–Ω–Ω—ã–º (true), –µ—Å–ª–∏ –µ–≥–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç ‚Äî –Ω–µ –Ω–æ–ª—å –∏–ª–∏ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –ø—É—Å—Ç—ã–º –æ–±—ä–µ–∫—Ç–æ–º. –°–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ, –æ–Ω–æ —Å—á–∏—Ç–∞–µ—Ç—Å—è –ª–æ–∂–Ω—ã–º (false), –µ—Å–ª–∏ —Ä–µ–∑—É–ª—å—Ç–∞—Ç ‚Äî –Ω–æ–ª—å –∏–ª–∏ –ø—É—Å—Ç–æ–π –æ–±—ä–µ–∫—Ç, –≤ —Ç–æ–º —á–∏—Å–ª–µ –∑–Ω–∞—á–µ–Ω–∏–µ none. –ü—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –≤—ã—Ä–∞–∂–µ–Ω–∏—è –±—É–¥–µ—Ç true, –µ—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –∏ false ‚Äî –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ.

–°—Ö–µ–º–∞—Ç–∏—á–Ω–æ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—é —Å if –º–æ–∂–Ω–æ –∑–∞–ø–∏—Å–∞—Ç—å —Ç–∞–∫:

–í–Ω—É—Ç—Ä–∏ –æ–¥–Ω–æ–≥–æ —É—Å–ª–æ–≤–∏—è –º–æ–∂–Ω–æ –≤—ã–ø–æ–ª–Ω—è—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏–π, —á—Ç–æ —Ä–∞—Å—à–∏—Ä—è–µ—Ç –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è —É—Å–ª–æ–≤–Ω—ã—Ö –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–π. –≠—Ç–æ –º–æ–∂–Ω–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å —Å –ø–æ–º–æ—â—å—é –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ and, not –∏ or:

üü¢ **and** ‚Äî –æ–∑–Ω–∞—á–∞–µ—Ç ¬´–ò¬ª –¥–ª—è –¥–≤—É—Ö —É—Å–ª–æ–≤–∏–π. –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç **tru**e, –µ—Å–ª–∏ –æ–±–∞ —É—Å–ª–æ–≤–∏—è –∏—Å—Ç–∏–Ω–Ω—ã, –∏ **false** ‚Äî –µ—Å–ª–∏ –ª–æ–∂–Ω—ã;

## –û–ø–µ—Ä–∞—Ç–æ—Ä if-else

–û–ø–µ—Ä–∞—Ç–æ—Ä **if-else** –≤ Python ‚Äî —ç—Ç–æ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç –≤—ã–ø–æ–ª–Ω–∏—Ç—å –æ–¥–∏–Ω –∏–∑ –¥–≤—É—Ö –±–ª–æ–∫–æ–≤ –∫–æ–¥–∞ –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ç–æ–≥–æ, –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —É—Å–ª–æ–≤–∏–µ (–≤—ã—Ä–∞–∂–µ–Ω–∏–µ –ø–æ—Å–ª–µ if) –∫–∞–∫ –∏—Å—Ç–∏–Ω–Ω–æ–µ –∏–ª–∏ –ª–æ–∂–Ω–æ–µ. –ï—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ **–∏—Å—Ç–∏–Ω–Ω–æ**, –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è **–±–ª–æ–∫ –∫–æ–¥–∞ –ø–æ–¥ if**. –ï—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ **–ª–æ–∂–Ω–æ** ‚Äî **else**.

## –í–ª–æ–∂–µ–Ω–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã if –∏ if-else
